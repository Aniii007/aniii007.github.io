for loop Syntax

for (initialization; condition; increment/decrement){
code to be executed
}
for (i=54; i>=31; i-=3){
document.write(i+ "<br>")
}
This will start from 54 and keep getting subtracted by 3 until value is 31.


if..else statement Syntax

if (condition){
code to be executed if condition is true
}
else {
code to be executed if condition is false
}
if (price>100){
document.write("Price is high")
}
else {
document.write("Price is cheap")
}


Push and Pull in array Syntax

variable.push("new") Adds "new" in the group at last.
variable.pull(indexnumber) Removes the last element in the array.

var you = new Array("cute", "caring", "human", "student")
you.push("stressed")
you.pull()
